AFINDUSTRIAL – Sitio Web Oficial

Repositorio del sitio web corporativo de AFINDUSTRIAL, empresa
especializada en el afilado industrial de cuchillas, sierras y
herramientas de corte en la Ciudad de México.

DESCRIPCIÓN GENERAL Este repositorio contiene el sitio web estático
utilizado para: - Presentación institucional. - Descripción de servicios
de afilado industrial. - Capacidades técnicas y procesos. - Solicitud de
cotización mediante formulario. - Ubicación y medios de contacto.

El proyecto está optimizado para motores de búsqueda (SEO técnico) y
utiliza una línea de diseño coherente en todas las páginas.

ESTRUCTURA DEL PROYECTO Archivos principales en la raíz del
repositorio: - index.html - empresa.html - afilado-industrial.html -
servicios.html - cotizacion.html - ubicacion.html - contacto.html -
sitemap.xml - robots.txt - img/ (carpeta de imágenes del sitio)

TECNOLOGÍAS UTILIZADAS - HTML5 - CSS3 - Google Fonts - Font Awesome -
FormSubmit (gestión de formularios sin backend) - Google Maps Embed

SEO INTEGRADO El sitio implementa los elementos esenciales para una
indexación eficiente: - Títulos y descripciones optimizadas por
página. - Enlaces canónicos. - Etiquetas Open Graph. - Etiquetas Twitter
Card. - Archivo robots.txt. - Archivo sitemap.xml.

FORMULARIOS Los formularios envían la información directamente a:
afindustrial@hotmail.com

ACTUALIZACIÓN DEL CONTENIDO 1. Modificar cualquier archivo HTML o
imagen. 2. Confirmar los cambios en GitHub. 3. El dominio reflejará los
cambios automáticamente.
