# AFINDUSTRIAL – Sitio Web Oficial
Soluciones profesionales en afilado industrial, mantenimiento de herramientas de corte y soporte técnico para procesos metalmecánicos y manufactureros.

Este repositorio contiene el código fuente del sitio web institucional de AFINDUSTRIAL.

---

## Perfil de la Organización

AFINDUSTRIAL ofrece servicios especializados de:

- Afilado de cuchillas industriales  
- Afilado de sierras circulares y de fricción  
- Rectificado de herramientas  
- Mantenimiento a maquinaria de corte  
- Fabricación y rehabilitación de rodillos formadores  

El sitio web funciona como plataforma central de comunicación, información técnica y recepción de solicitudes de servicio.

---

## Alcance del Repositorio

Incluye:

- Estructura completa del sitio  
- Archivos HTML corporativos  
- Formularios operativos mediante FormSubmit  
- Componentes visuales e imagen institucional  
- Configuración SEO para indexación en Google  

---

## Estructura del Proyecto

Archivos principales ubicados en la raíz:

- index.html  
- empresa.html  
- afilado-industrial.html  
- servicios.html  
- cotizacion.html  
- ubicacion.html  
- contacto.html  
- sitemap.xml  
- robots.txt  
- img/ (recursos gráficos)

---

## Tecnologías Utilizadas

- HTML5  
- CSS3  
- Google Fonts: Montserrat  
- Font Awesome  
- Google Maps Embed  
- FormSubmit (formularios sin backend)

---

## SEO y Optimización

El sitio cuenta con:

- Títulos y descripciones optimizadas por página  
- Canonical tags por documento  
- Open Graph para vistas enriquecidas  
- Twitter Cards  
- Archivo robots.txt  
- Archivo sitemap.xml estructurado  

Orientado a mejorar posicionamiento para búsquedas del sector industrial.

---

## Formularios de Contacto

Los formularios envían la información directamente al correo corporativo:

**afindustrial@hotmail.com**

Implementados mediante la plataforma FormSubmit, sin necesidad de servidores adicionales.

## Mantenimiento y Actualización

Pasos para actualizar el sitio:

1. Editar cualquier archivo HTML o recurso.  
2. Realizar commit y push al repositorio.  
3. El dominio asociado reflejará los cambios automáticamente.  

Arquitectura 100% estática y de alta disponibilidad.
