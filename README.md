# AFINDUSTRIAL – Sitio Web Oficial

Repositorio del sitio web corporativo de **AFINDUSTRIAL**, empresa dedicada al afilado de cuchillas, sierras y herramientas de corte para el sector industrial en la Ciudad de México.

---

## Descripción general

Este proyecto contiene el código fuente del sitio estático utilizado para presentar:

- Información institucional.
- Servicios de afilado industrial.
- Capacidades técnicas.
- Formulario de cotización y contacto.
- Ubicación y horarios de atención.

El sitio está optimizado para indexación en buscadores (SEO técnico) y funciona sin backend mediante formularios gestionados por FormSubmit.

---

## Tecnologías utilizadas

- HTML5  
- CSS3  
- Google Fonts  
- Font Awesome  
- FormSubmit (envío de formularios)  
- Google Maps Embed  

---

## SEO incorporado

El sitio incluye:

- Etiquetas `<title>` y `<meta description>` por página.
- Enlaces canónicos.
- Etiquetas Open Graph y Twitter Card.
- Archivo `robots.txt` para permitir la indexación.
- Archivo `sitemap.xml` con todas las rutas principales.

---

## Formularios

Los formularios de contacto y cotización envían la información directamente a:

**afindustrial@hotmail.com**

Sin necesidad de backend adicional.

---

## Actualización del contenido

Para actualizar el sitio:

1. Modificar cualquier archivo HTML o imagen.
2. Confirmar los cambios en GitHub.
3. El dominio sincronizado reflejará la actualización.
